The dim

The dim is a variable value that you can change. The following codes define a dim "dimName" of type "dimDataType" without initialize:

dim <dimType> <dimName>

And a dim "dimName" fo type "dimType" with initialize "dimInitializeValue":

dim <dimType> <dimName>=<dimInitializeValue>

If you want to create more dims in a single line, use this:

dim <dimType> <dimName1>[=<dimInitializeValue1>], <dimName2>[=<dimInitializeValue2>], ...

Example

Example 1

function int dims(void)
    dim int a=123, b=456
    dim int c
    c = a+b
    return c
end
