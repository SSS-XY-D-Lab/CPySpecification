Dim

The dim

The dim is a variable value that you can change. The following codes define a dim "dimName" of type "dimType" without initialize:

dim <dimType> <dimName>

And with initialize "dimInitializeValue":

dim <dimType> <dimName>=<dimInitializeValue>

If you want to create more dims in a single line, use this:

dim <dimType> <dimName1>[=<dimInitializeValue1>], <dimName2>[=<dimInitializeValue2>], ...

Const dim

The "const" keyword means that the dim can't be change, just like the following:

const <constDimType> <constDimName1>=<value1>, <constDimName2>=<value2>, ...

Pointer

The pointer save the address of a dim.Use "&" to get the address of a dim, just like the following:

dim int a
dim int* p=&a

And the pointer to a pointer:

dim int a
dim int* p1=&a
dim int* p2=&p1

To visit what pointer is pointed to, use the "*":

dim int a=1, b=2, c
dim int* pa=&a, pb=&b
c=*pa+*pb; # The same as c=a+b

Array

An array is a series of dims.The following codes define an array "arrayName" of size "arraySize" with dim type "dimType" without initialize:

dim <dimType> <arrayName>[arraySize]

And with initialize datas:

dim <dimType> <arrayName>[arraySize]={<value1>, <value2>, ...}

CPy only support 1-D array now.To visit one dim in array, use "[index]".Index start at 0, end at ()index-1).For example:

dim int array[2]={1,2}
dim int arr0=array[0] # Get 1.
dim int arr1=array[1] # Get 2.
#dim int arr2=array[2] # DANGEROUS!!! Don't write like this, for you don't know what you will get!

If you try to get the pointer of the array, you will get the pointer to the first dim in the array.

dim int array[2]={1,2}
dim int* parray=&array

Example

Example 1

function int dims(void)
    dim int a=123, b=456
    dim int c
    c = a+b
    return c
end

Example 2

function int array(void)
    dim int array[2]={1,2}
    dim int ret
    ret = array[0]+array[1]
    return ret
end
