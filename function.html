Function

Define a function

The following codes define a function:

function <returnDataType> <functionName>([<arg1DataType> <arg1Name>[, <arg2DataType> <arg2Name>[, ...]]])
    <TODO: Your codes>
end

If your function have no arg:

function <returnDataType> <functionName>(void)
    <TODO: Your codes>
end

Notice: the line of the function defination(With "function" keyword at the begining) MUST BE IN A SINGLE LINE.

The following codes are WRONG:

function <returnDataType> <functionName>
  (void)
    <I am wrong!>
end

Function Return

A line with single statement "return" in the function can let the function return, with(or without) a return value, just like following:

return <returnData>

The returnData must match the returnDataType.

Example

Example 1

function int main(void)
    return 0
end

Example 2

function int add(int a, int b)
    return (a+b)
end
